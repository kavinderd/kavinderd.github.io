Some C++ knowledge:

  • the mutable keyword in C++ allows for a value of something to be changed even by a constant reference
  • the friend keyword allows for access to the private and protected members of a class:

    • Example: class A {

            public updateB(B& b) { 
          // THis is possible because A is a friend of B
          b.m_myInt++
      }
      }

      class B
      {
      friend class A;
      private:
      int m_myInt;
      }

    • friend relationships are not commutative or transitive
